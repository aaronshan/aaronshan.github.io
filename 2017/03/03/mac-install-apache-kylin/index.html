mac osx单机上安装apache-kylin   2017-03-03


最近开始调研apache-kylin，关于kylin的介绍往上有很多，本文就不再过多介绍，只讲述如何在mac osx配置测试用的kylin环境。
需要安装及部署的软件包括：hadoop,mysql,hive,zookeeper,hbase,kylin。本篇主要讲述如下版本的安装工作：

  • hadoop-2.7.3
  • mysql-5.6.31
  • hive-2.1.1
  • zookeeper-3.4.8
  • hbase-1.3.0
  • kylin-1.6.0-hbase1.x
